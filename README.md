# 自动生成周报

## 安装

`npm i -g wtf-weekly`

## 用法

1. `cd a/git/project/dir`
2. `ww`  or `wtf-weekly`
3. 得到周报
    ```
    周一
    fix phone verify
    update styles
    rename
    add svg

    周二
    补全用例
    refactor
    typo
    remove debug code
    remove unused

    周三
    update styles

    周四
    complete avatar module

    周五
    fix testing
    remove cache

    周六
    无

    周日
    无
    ```
